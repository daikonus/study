# Vuejs開発環境の環境構築
## パッケージ管理
>npm -v
6.12.1

## Nodejs
>node -v
v12.13.1

## vue-cliのインストール
>npm install -g @vue/cli
>vue -V
@vue/cli 4.1.1

## プロジェクトの初期化
vue init webpack <ProjectName>

## vueの実行
npm run serve

Top画面
http://localhost:8080
